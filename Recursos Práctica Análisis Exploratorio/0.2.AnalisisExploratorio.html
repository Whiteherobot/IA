Definiciones de anlisis de datos

  • El anlisis de datos es un proceso de recoleccin, limpieza, transformacin y modelado de datos con el objetivo de descubrir informacin til, informar conclusiones y respaldar la toma de decisiones.

  • El anlisis de datos tiene mltiples facetas y enfoques, que abarcan diversas tcnicas bajo una variedad de nombres como minera de datos, business intelligence, aprendizaje automtico, anlisis del bigdata, etc.

  • El anlisis de datos es la piedra angular en diferentes dominios de negocios y de las ciencias.

  • El anlisis exploratorio de datos es una etapa fundamental en el proceso de anlisis de datos para comprender la esencia de los datos, identificar problemas de calidad y definir la direccin que tomar el anlisis posterior.

Tipos de variables

1. Categricas: Las variables categricas indican el tipo o categora de una observacin. Por lo tanto, las variables categricas son variables cualitativas y en la mayora de conjuntos de datos estn representadas por un valor no numrico.

1.1 Nominales: la categora indica la identidad del objetivo. No hay un grado o diferencia entre las categoras, lo que se enfatiza es el nombre. Algunos ejemplos de variables categricas nominales son: sexo, pas, color de ojos, etc.

1.2 Ordinales: la categora indica un orden. Algunos ejemplos de variables categricas ordinales son: nivel de educacin (primaria, secundaria, superior), las escalas de las encuestas de satisfaccin (insatisfecho, neutro, satisfecho), calificacin cualitativa (insuficiente, buena, muy buena, sobresaliente, muy sobresaliente), etc.

2. Numricas: Las variables numricas tienen valores que describen una cantidad medible. Las variables numricas son variables cuantitativas.

2.1 Continuas: estas son variables que son cuantitativas y pueden medirse a lo largo de una secuencia o un rango de valores. Existen dos tipos de variables continuas: las variables de intervalo y las variables de relaciones. Las variables de intervalo pueden tener cualquier valor dentro de un rango de valores. Algunos ejemplos son temperatura o tiempo. Las variables de relaciones son las variables de intervalo especiales donde un valor de cero (0) significa que no hay ninguna variable. Entre los ejemplos se incluyen ingresos o el volumen de ventas.

2.2 Discretas: estos tipos de variables continuas son cuantitativos, pero tienen un valor especfico de un conjunto de valores finito. Los ejemplos incluyen el nmero de sensores habilitados en una red, el nmero de automviles en un estacionamiento, la cantidad de visitas a una pgina web, la cantidad de llamadas recibidas, etc.

Actividades del anlisis exploratorio

Las actividades clave en un anlisis exploratorio de datos incluyen:

  • Carga de Datos: Importar los datos desde diferentes fuentes, como archivos CSV, hojas de clculo, pginas web, bases de datos, etc.

  • Exploracin de Datos Iniciales: Comprender la estructura de los datos, tamao, cantidad de variables y observaciones, tipos de variables y la calidad de los datos (verificar si hay valores faltantes, duplicados o atpicos).

  • Resumen Estadstico: Calcular estadsticas descriptivas bsicas, como la media, mediana, desviacin estndar, percentiles, para comprender la distribucin y la variabilidad de los datos.

  • Visualizacin de Datos: Crear grficos y visualizaciones para representar los datos de manera efectiva. Esto puede incluir histogramas, diagramas de dispersin, grficos de barras y diagramas de caja, entre otros.

  • Anlisis de Variables Categricas: Si existen variables categricas, realizar anlisis de frecuencia para comprender la distribucin de categoras.

  • Anlisis de Correlacin: Evaluar las relaciones entre las variables mediante el clculo de correlaciones.

  • Manejo de Datos Faltantes: Decidir cmo manejar los valores faltantes, que puede incluir imputacin (sustitucin), eliminacin o anlisis de su impacto en los resultados.

  • Anlisis de Outliers: Identificar y comprender los valores atpicos que pueden afectar el anlisis y la interpretacin de los resultados.

  • Segmentacin de Datos: Si es relevante, dividir el conjunto de datos en segmentos o grupos con caractersticas similares para un anlisis ms detallado.

  • Generacin de Hiptesis: Basado en la exploracin de datos, formular hiptesis iniciales sobre relaciones o patrones que se puedan investigar ms adelante.

  • Presentacin de Resultados: Comunicar hallazgos iniciales a travs de informes o presentaciones que resuman las observaciones y los posibles pasos siguientes en el anlisis.

<std:> esta es la desviacin estndar de la distribucin.

75%: este es el valor para el tercer cuartil. El 75 % de los valores se encuentran en o por debajo de este valor.

<max:> este es el valor ms alto de la distribucin.

Qu conclusiones puedes indicar a partir de las estadsticas?

Por ejemplo:

Edad Media: La edad media de los clientes del banco es de 35 aos. Esto significa que, en promedio, los clientes tienen alrededor de 35 aos de edad.

Edad Mnima: La edad mnima registrada entre los clientes es de 19 aos. Esto indica que el banco tiene clientes jvenes, incluyendo a personas que recin comienzan su vida financiera.

Edad Mxima: La edad mxima registrada entre los clientes es de 75 aos. Esto sugiere que el banco tambin atrae a clientes mayores que pueden estar planificando su jubilacin o gestionando sus finanzas en la edad adulta.

Diversidad de Edades: La diferencia entre la edad mnima y mxima (75 - 19 = 56 aos) es bastante amplia (desviacin estandar es de 11.37). Esto refleja una diversidad de edades en la base de clientes del banco, lo que puede requerir una variedad de servicios y productos adaptados a diferentes grupos demogrficos.

Poblacin Equilibrada: En trminos generales, la edad media de 35 aos sugiere que la poblacin de clientes del banco tiende a ser equilibrada en cuanto a edades, sin una inclinacin significativa hacia un grupo etario particular.

Estas conclusiones pueden ser tiles para el banco a la hora de tomar decisiones estratgicas sobre sus servicios y productos. Por ejemplo, podran considerar la introduccin de productos financieros especficos para jvenes o para personas mayores, o disear campaas de marketing dirigidas a diferentes grupos de edad. Tambin podran utilizar esta informacin para mejorar la experiencia del cliente y la personalizacin de servicios en funcin de la edad.

import plotly.graph_objs as go
import numpy as np
import plotly.express as px
from mpl_toolkits.mplot3d import Axes3D

# Genera datos de ejemplo (reemplaza esto con tus propios datos)
#np.random.seed(0)
#data = np.random.rand(100, 3)
#np.random.rand(200)

# Selecciona las tres columnas que deseas convertir
columnas_seleccionadas = dataframe_scaled_minmax[['PLAZOMESESCREDITO', 'MONTOCREDITO', 'TASAPAGO']]

# Convierte las columnas seleccionadas en un numpy.ndarray
data = columnas_seleccionadas.to_numpy()

# Define un umbral (de distancia euclidiana) para identificar outliers (puedes ajustar esto segn tus necesidades)
umbral = 1.2

# Calcula la distancia euclidiana desde el origen (centro) para cada punto de datos
distancias = np.linalg.norm(data, axis=1)

# Identifica los ndices de los outliers
outliers_indices = np.where(distancias > umbral)[0]

# Separa los datos en dos grupos: puntos normales y outliers
puntos_normales = data[distancias <= umbral]
outliers = data[outliers_indices]

# Crea una figura 3D interactiva
fig = go.Figure()

# Plotea los puntos de datos normales (dentro de la esfera)
fig.add_trace(go.Scatter3d(
    x=puntos_normales[:, 0],
    y=puntos_normales[:, 1],
    z=puntos_normales[:, 2],
    mode='markers',
    marker={'color':'blue', 'size':4},
    name='Puntos Normales'
))

# Plotea los outliers (fuera de la esfera)
fig.add_trace(go.Scatter3d(
    x=outliers[:, 0],
    y=outliers[:, 1],
    z=outliers[:, 2],
    mode='markers',
    marker={'color':'red', 'size':4},
    name='Outliers'
))

# Crea una esfera que excluye los outliers
u = np.linspace(0, 2 * np.pi, 100)
v = np.linspace(0, np.pi, 100)
x = umbral * np.outer(np.cos(u), np.sin(v))
y = umbral * np.outer(np.sin(u), np.sin(v))
z = umbral * np.outer(np.ones_like(u), np.cos(v))

fig.add_trace(go.Surface(x=x, y=y, z=z, colorscale='greens', opacity=0.3, showscale=False, name='Esfera Umbral'))

# Configura el diseo de la figura
fig.update_layout(
    scene={
        'xaxis_title':'X: PLAZOMESESCREDITO', #EJE X
        'yaxis_title':'Y: MONTOCREDITO', #EJE Y
        'zaxis_title':'Z: TASAPAGO' #EJE Z
    },
    scene_camera={
        'center':{'x':0, 'y':0, 'z':0},
        'eye':{'x':1.25, 'y':1.25, 'z':1.25}
    }
)
# Guarda la figura interactica
